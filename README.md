# Rule

- 일주일에 한번 스터디합니다.
- 스터디의 주기는 짧습니다. ('객체지향의 사실과 오해'는 3주를 생각하고 있습니다.)
- 주차별 공부한 내역은 MD 파일로 전달하여 업로드합니다.
- 매주 함께 생각할 내용이 있다면 [Issue](https://github.com/JNU-econovation/java-ggun/issues)에 업로드합니다.
